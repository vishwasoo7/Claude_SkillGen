# 🌾 Farming Skills

Custom-built Claude skills for the **Rangpur Natural Farm**, Keshod Taluka, Junagadh, Gujarat.
All skills are calibrated for Saurashtra/Gujarat soil, climate, and farming context.

| Skill | Lines | Description |
|-------|-------|-------------|
| [`agstack-farming`](./agstack-farming/SKILL.md) | 274 | AgStack Foundation digital agriculture advisor — ETo, pest risk, Gujarat crop calendar |
| [`vedic-farming`](./vedic-farming/SKILL.md) | 117 | ZBNF / Subhash Palekar preparations — Jeevamrut, Beejamrut, Panchgavya, lunar calendar |
| [`soil-test-interpreter`](./soil-test-interpreter/SKILL.md) | 140 | Indian soil lab report reader → organic amendment prescriptions per Vigha |

## Status
✅ All 3 skills are production-ready and validated for the VP Farm Project.

## Farm Context
- **Size:** 1.2 acres (3 Vigha)
- **Phase 1 Start:** April–May 2026
- **Method:** Subhash Palekar Natural Farming (ZBNF) + organic transition
- **Location:** Rangpur village, Keshod Taluka, Junagadh, Gujarat

## How to Install
Copy any skill folder to `/mnt/skills/user/` in your Claude environment.
