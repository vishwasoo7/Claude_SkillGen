# Farm Journal
## Structured Observation and Activity Log
### Rangpur Village, Keshod Taluka, Junagadh, Gujarat

Your farm journal is your most important long-term tool. Every observation and action
logged here builds a season-over-season knowledge base that no other tool can provide.
In 3 years, it will tell you more about your land than any government report.

All entries use the structured templates below. Use whichever template fits your observation.
Multiple templates can be filled in a single conversation.

---

## HOW TO USE THIS SKILL

**Three ways to log:**

**A) Conversational** -- Describe what you saw or did in natural language.
"Today I applied Jeevamrut to the north part, soil smelled good, found 4 earthworms per square foot"
I will convert it into a structured entry and confirm.

**B) Template fill** -- Ask for a specific template and fill in the fields.
"Give me the soil observation template"

**C) Review** -- Ask to see, compare, or summarize past entries.
"What was the earthworm count last Kharif?" or "Summarize last season's pest events"

## ENTRY TYPES AND TEMPLATES

---

### TYPE 1 - SOIL OBSERVATION

Use every 2-4 weeks, and always before and after each Jeevamrut batch.

```
[SOIL-OBS]
Date: YYYY-MM-DD
Tithi (optional): [Tithi name and Paksha]
Location on farm: [North / South / East / West / All / specific parcel]
Season: [Transition / Kharif / Rabi / Summer]
Days since last Jeevamrut: [number]

PHYSICAL CHECK
  Soil color (dry): [dark brown / medium brown / light / grey]
  Soil color (moist): [dark / medium / light]
  Texture when pressed: [crumbles / sticky / hard / powdery]
  Smell: [earthy-pleasant / neutral / sour / chemical / no smell]
  Surface crust: [none / light / moderate / hard]
  Moisture at 3 inches depth: [dry / slightly moist / moist / wet]

BIOLOGICAL CHECK
  Earthworm count: [number found in 1 sq. ft x 6 inch dig]
  Location of dig: [describe]
  Other life observed: [beetles / white grubs / other / none]
  Fungal threads visible (white): [yes / no]
  Root condition (if uprooted plant): [healthy white / brown / stunted]

WATER BEHAVIOR
  After irrigation, water absorbed in: [minutes]
  Surface pooling: [yes / no]
  Cracks visible: [yes / no -- describe size]

NOTES:
[Any other observations, comparisons to last check, unusual things]
```

### TYPE 3 - CROP OBSERVATION

Use weekly during growing season. Capture one entry per crop type.

```

Parcel: [describe location]
Days after sowing (DAS): [number]
Growth stage: [germination / seedling / vegetative / flowering / pod fill / maturity]

PLANT APPEARANCE
  Leaf color: [dark green / medium green / pale / yellow / brown patches]
  Leaf condition: [healthy / curling / spots / holes / dropping]
  Stem: [healthy / weak / discolored / lodged]
  Canopy cover: [full / patchy / thin]

GERMINATION (if before 30 DAS)
  Germination rate estimate: [%]
  Patchy areas: [yes / no -- describe location and size]
  Likely cause of patches: [herbicide residue / poor seed / dry zone / other]

PEST / DISEASE
  Any pest visible: [yes / no]
  Pest type: [describe]
  Severity: [light / moderate / severe]
  Affected area: [% of parcel]
  Action taken: [remedy applied / monitoring / none yet]

ROOT CHECK (if any plant uprooted)
  Root color: [white / brown]
  Root depth: [estimated cm]
  Root branching: [good / poor]
  Nodules visible (legumes only): [yes / no / many / few]

NOTES:
[Comparison to last observation, anything notable]
```

### TYPE 4 - IRRIGATION / WATER LOG

Use after every irrigation event and after significant rainfall.

```
[IRRIGATION-LOG]

IF BOREWELL
  Duration: [hours and minutes]
  Motor running well: [yes / no -- note any issues]
  Estimated volume (if known): [litres]
  Power source: [grid / generator / other]
  Any failure or interruption: [yes / no -- describe]

IF RAINFALL
  Duration: [approximate hours]
  Intensity: [light drizzle / moderate / heavy]
  Estimated depth: [mm if rain gauge available, otherwise light/medium/heavy]

SOIL RESPONSE
  Water reached parcel evenly: [yes / no]
  Runoff observed: [yes / no -- where]
  Puddles: [yes / no -- how long before absorbed]
  Soil surface after drying: [cracked / smooth / mulched/protected]

CROP RESPONSE
  Visible improvement after irrigation: [describe or none noted]
  Any stress signs before irrigation: [wilting / leaf curl / describe]

NOTES:
[Water use trend, concerns about availability, borewell behavior]
```

### TYPE 5 - PEST / DISEASE EVENT

Use immediately on first sighting of any pest or disease.

```
[PEST-EVENT]
Date: YYYY-MM-DD
Crop affected: [name]
Parcel: [describe]
DAS at time of sighting: [number]

PEST / DISEASE IDENTIFICATION
  What you see: [describe in your own words]
  Likely pest: [aphid / whitefly / caterpillar / borer / mite / fungal / unknown]
  Confidence: [sure / likely / unsure]
  Spread: [single plant / cluster / rows / scattered / spreading]
  Severity: [1 - 5 scale: 1 = few plants, 5 = major outbreak]

IMMEDIATE ACTION TAKEN
  Remedy chosen: [Neemastra / Brahmastra / Agniastra / Dashaparni / none yet]
  Quantity applied: [litres per Vigha]
  Application time: [morning / evening]
  Tithi: [optional]

FOLLOW-UP CHECK (fill after 5-7 days)
  Follow-up date: [YYYY-MM-DD]
  Pest still present: [yes / reduced / gone]
  Second application needed: [yes / no]
  Outcome: [describe]

NOTES:
[Any pattern from last season? Same pest, same timing?]
```

## IMPORTANT NOTES

- Date format: always use YYYY-MM-DD for sorting and comparison
- Use short descriptive notes -- don't over-explain. Consistency matters more than detail.
- Log even when nothing notable happened. "No pests, soil normal, good moisture" is valuable.
- The comparison between seasons is where the real value emerges. Log consistently.
- Your journal is private but can be shared as demonstration evidence. Structure it accordingly.
